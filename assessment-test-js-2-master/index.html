CSS3, Javascript & jQuery Test

This test is meant to help gauge your current skill level with basic front-end programming (with JavaScript).

The test consists of 5 questions that total to 100 points. Your aim should be to score 80 or higher. There is also a bonus question worth 10 points as a part of question #4.

Tips:

  • Use Chrome's DevTools (Console, etc)... It is your friend.
  • If you are struggling with a solution, move onto the next question
  • There are numerous "correct" ways of solving any coding problem. Feel free to use online resources to help you come up with solutions.

Rules:

  • Follow the guidelines in the code about what part of each file you can update.
  • Googling is allowed to help recall syntax, etc.
  • The teachers floating about can help explain questions but not help you solve your coding issues or questions
